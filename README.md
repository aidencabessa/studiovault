# StudioVault

Free Luau script library + AI script fixer for Roblox developers.

## Local Development

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this folder to a GitHub repo
2. Go to vercel.com → Add New Project → Import repo
3. Click Deploy (Vercel auto-detects Vite)

## Stack

- React 18
- Vite 5
- No UI libraries — all custom CSS
- AI fixer powered by Anthropic API
